# Bridge Runner Game

A simple endless runner game made with Python and Pygame.

## Controls
- **Space**: Jump
- **Space (after Game Over)**: Restart

## Setup
1. Ensure Python is installed.
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## Audio
Place your `scream.wav` file in this directory to enable the Game Over sound effect.

## Assets
The game uses the following asset files (generated automatically):
- `player.png`
- `bridge.png`
- `fire.png`
- `background.png`

You can replace these images with your own to customize the look!
